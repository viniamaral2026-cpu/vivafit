# **App Name**: VivaFit MVP

## Core Features:

- Premium Content: Enable access to premium workout content for subscribers.
- User Authentication: Secure user accounts with login and registration.
- Subscription Management: Allow users to manage their subscription status.
- Payment Gateway Integration: Integrate with Pagseguro for processing payments.
- Dashboard Overview: Admin dashboard showing key metrics like users, payments, and content.
- Content Management: Allow admins to manage premium workout content and ads.

## Style Guidelines:

- Primary color: Energetic violet (#A076F9) to represent vitality.
- Background color: Light gray (#F5F3F6), a desaturated variant of the primary violet.
- Accent color: Soft blue (#74A2F2), to complement the violet with a touch of calm.
- Body and headline font: 'PT Sans' for a clean, modern and friendly appearance.
- Use clean, minimalist icons related to fitness activities.
- Simple, intuitive layout with clear calls to action.
- Subtle transitions and animations for a smooth user experience.
